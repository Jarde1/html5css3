One Page Site

Ideal para negócios de pequeno e médio porte o "one page site" é um site onde você inclui todas as informações que o visitante precisa de forma objetiva e criativa em uma única página.

Wireframe

Wireframe é uma representação visual da estrutura básica de uma página. Ele deve ser feito da mesma maneira mais Simples possível, mostrando apenas o essencial, como uma espécie de rascunho, sem cores ou imagens. Clique no botão ou escaneie o QR Code para obter o wireframe desta página.
